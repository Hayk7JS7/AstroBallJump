# AstroBallJump

## Project Overview

AstroBallJump is a project created for a technical interview, featuring a solar system visualization and ball bouncing simulation.

## Setup Instructions

1. **Clone the Repository:**
   ```bash
   git clone https://github.com/Hayk7JS7/AstroBallJump.git

2. **Navigate to the Project Directory:**
cd AstroBallJump

3. **Install Dependencies:**
    # If you're using npm
    npm install
    
    # If you're using yarn
    yarn

4. **Run the Project:**
    # If you're using npm
    npm run dev
    
    # If you're using yarn
    yarn run dev

5. **Open in Browser:**
    Open your web browser and navigate to http://localhost:3000 to view the project.

## Project Details

AstroBallJump is an engaging project designed for a technical interview, showcasing a captivating combination of a solar system visualization and a dynamic ball bouncing simulation. It aims to demonstrate proficiency in web development, particularly in the realms of 3D rendering and animation.

### Solar System Visualization

The solar system visualization component provides an interactive and visually appealing representation of celestial bodies. Leveraging the power of Three.js, a popular JavaScript library for creating 3D graphics, the solar system comes to life with accurate planetary movements, textures, and orbital rotations.

### Ball Bouncing Simulation

Integrated seamlessly with the solar system, the ball bouncing simulation introduces an interactive element. Users can initiate ball bouncing interactions within the solar system, creating an engaging and dynamic experience. This simulation serves as a testament to the project's versatility and creative implementation.

### Technologies Used

- **Three.js:** The project heavily relies on Three.js, a JavaScript library that simplifies the creation of 3D graphics in the browser. It provides a robust set of tools for rendering and animating 3D scenes.

### How to Interact

1. **Explore the Solar System:**
   - Navigate through the solar system using intuitive controls.
   - Observe the accurate movements of planets and their orbital rotations.

2. **Engage in Ball Bouncing:**
   - Initiate ball bouncing simulations within the solar system.
   - Observe the dynamic interactions between the bouncing ball and celestial bodies.
